# Task 2 - Docker Basics

## 📌 Objective
Learn basic Docker commands and container usage.

## 🔧 Steps Performed

1. Checked Docker version:
   docker --version

2. Pulled nginx image:
   docker pull nginx

3. Ran container:
   docker run -d -p 8080:80 nginx

4. Checked running containers:
   docker ps
