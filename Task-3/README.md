# Task 3 - Dockerfile Project

## 📌 Objective
Create a custom Docker image using a Dockerfile.

## 🔧 Steps Performed

1. Created index.html
2. Created Dockerfile using nginx
3. Built image:
   docker build -t my-app .

4. Ran container:
   docker run -d -p 8081:80 my-app

5. Accessed application in browser

## 📚 Conclusion
Learned how to build and run Docker images using Dockerfile.
